🚀 加油站销量预测系统
基于 Facebook Prophet 时间序列模型，预测加油站未来52周销量。结合促销、节假日等业务因素，为库存管理和运营调度提供数据支持。

📝 使用方法
1. 数据准备
创建Excel文件 加油站周销量真实数据.xlsx，包含以下列：

ds：周次，格式如"2023-W01"

y：销量（单位：升）

is_promo：是否促销周（0/1）

is_holiday：是否节假日周（0/1）

2. 运行程序
bash
# 基础运行
python main.py

# 程序会输出：
# === 两年数据版预测完成 ===
# 训练数据：2023 + 2024 共104周
# 2025年预计总销量：41280
# 第一周（W01）预测：1100
# 最后一周（W52）预测：1069
3. 自定义设置
修改 main.py 中的促销周设置：

python
# 找到这行注释，取消注释并修改：
# future.loc[future['ds'].dt.week.isin([8,18,28,38]), 'is_promo'] = 1
# 改为：
promo_weeks = [8, 18, 28, 38]  # 自定义促销周
future.loc[future['ds'].dt.week.isin(promo_weeks), 'is_promo'] = 1
